# وارد کردن دیتا در جدول

در پوشه `js` یک فایل با نام `data.txt` وجود دارد که دیتایی با فرمت `json` در آن وجود دارد که باید آن را در نرم افزار هایی مانند `xamp` - `wamp` - `laragon`
یا هر نرم افزاری که این قابلیت را دارد اجرا کنید.

 که پیش از اجرا نمونه آدرس در لوکال هاست به این 
 صورت میشود
 
 
 ```
 localhost/wordpress/data.txt
 ```
بعد از اجرا در لوکال هاست شما باید خطایی که برای مرورگر وجود دارد را نیز حل کنید 

![GitHub Logo](https://s16.picofile.com/file/8426223542/1.png)

برای حل این مشکل شما به دو چیز نیاز دارید 

- مرورگر `chrome`
- افزونه `Allow CORS: Access-Control-Allow-Origin`

 - [لینک دانلود افزونه ( مخصوص کروم )](https://chrome.google.com/webstore/detail/allow-cors-access-control/lhobafahddgcelffkeicbaginigeejlf)
 
- بعد از نصب و فعالسازی میتوانید داده را به صورت `Json` دریافت کنید
